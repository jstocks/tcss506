 FDA OpenAPI - Query Tool


  This site was created by Jeff Stockman & Sriba Rajendran as a Capstone Project for the Graduate Certificate Program
  in Software Development Engineering at the Unversity of Washington - Tacoma.

Requirements:
1) Hosted on AWS EC2 Instance
2) Flask Server
3) NGINX reverse proxy to Gunicorn WSGI
4) User Registration & Login via PostgreSQL Database
5) Docker Container Deployment

Features:
1) Uses FDA OpenAPI JSON dataset found here
2) Customized Filters and search capability
3) Email subscription to weekly updates implemented using the Producer - Consumer pattern with AWS SQS

  The source code can be found on GitHub.
